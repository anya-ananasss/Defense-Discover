# Название проекта
Гибридная игра "T. P. Defense"
# Описание проекта
"T.P. Defense" — гибридная веб-игра, сочетающая механику Tower Defense и викторину (quiz). Игроку необходимо отбить как можно больше волн врагов. Чтобы построить новую башню или улучшить существующую, игрок должен ответить на вопрос викторины.
